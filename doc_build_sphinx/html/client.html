Client module documentation¶

Клиентское приложение для обмена сообщениями. Осуществляет отправку сообщений другим пользователям, подключенным к серверу. Поддерживает аргументы командной строки:

python client.py {имя сервера} {порт}

  1. {имя сервера} - адрес сервера для подключения.

  2. {порт} - порт для подключения.

Пример запуска из основной директории:

  • python client/client.py

Запуск приложения с параметрами по умолчанию.
